# Food-Ordering-System

![1](https://github.com/mehrapallavi/Food-Ordering-System/assets/117376335/3c26449b-96ab-43a1-9fc1-61df7d74a418)


## Food Ordering System is a user-friendly application that is designed to use in the food industry where customers can easily browse through different food items and can easily place orders with just a few clicks.This is a Food Ordering System implemented in Java, featuring a customer login , admin login register page , a dashboard screen and various other functionalities. The application utilizes SQL database.

# Features
## 1.Authentication  
## • Login
## • SignUp
## 2.Orders
## • Food Menu
## • Food Cart 
## • Generates Bill
## 3.A solid database back end.
## 4.Work progresses quickly. Simple information updates.
## 5.The system provides searching facilities based on various factors such as item category, order, and shopping cart.
